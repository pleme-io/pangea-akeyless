# pangea-akeyless

Akeyless provider resources for the Pangea infrastructure DSL. All resource
code is **auto-generated** by `pangea-forge` via `iac-forge-cli`. Manual edits
to resource files will be overwritten on the next generation run.

## Structure

```
bin/
  generate                   # Regeneration script
lib/
  pangea-akeyless.rb         # Entry point (generated — requires all resources)
  pangea-akeyless/version.rb # Version constant (manual)
  pangea/
    types/akeyless_types.rb  # Provider-specific Dry::Types (manual)
    resources/
      akeyless.rb            # Aggregator module (generated)
      akeyless_*/            # 238 resource directories (generated)
        types.rb             # Dry::Struct attributes class
        resource.rb          # ResourceBuilder DSL
spec/
  spec_helper.rb             # Test setup (manual)
  resources/                 # RSpec stubs (generated)
```

## Generated vs Manual

| File | Source |
|------|--------|
| `lib/pangea/resources/akeyless_*/` | Generated by pangea-forge |
| `lib/pangea-akeyless.rb` | Generated (require file) |
| `lib/pangea/resources/akeyless.rb` | Generated (aggregator) |
| `spec/resources/*.rb` | Generated (RSpec stubs) |
| `lib/pangea-akeyless/version.rb` | Manual |
| `lib/pangea/types/akeyless_types.rb` | Manual |
| `spec/spec_helper.rb` | Manual |
| `Gemfile`, `*.gemspec`, `flake.nix` | Manual |

## Regeneration

```sh
# Using the bundled script (uses default paths)
bin/generate

# With custom paths
AKEYLESS_OPENAPI_SPEC=path/to/openapi.yaml \
AKEYLESS_RESOURCES=path/to/resources/ \
AKEYLESS_PROVIDER=path/to/provider.toml \
IAC_FORGE=path/to/iac-forge \
bin/generate
```

Default paths:
- Spec: `~/code/github/pleme-io/akeyless-go/api/openapi.yaml`
- Resources: `~/code/github/pleme-io/akeyless-terraform-resources/resources`
- Provider: `~/code/github/pleme-io/akeyless-terraform-resources/provider.toml`

## Resource count

- **238 resources** (auth methods, dynamic secrets, rotated secrets, targets,
  keys, certificates, gateway config, roles, event forwarders, misc)
